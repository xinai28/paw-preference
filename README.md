Find your favorite kitty with this interactive cat swiping app! Like or dislike cats and see your preferences summarized.

**Live Demo:** [https://paw-preference-v2.vercel.app](https://paw-preference-v2.vercel.app)

---

## Features

- Swipe through cats (like or dislike)
- Track your liked and disliked cats
- Summary page showing your results
- Restart and refresh to discover new cats
- Dynamic cat images fetched from [Cataas API](https://cataas.com/)


## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite for bundling
- Cataas API for cat images
- Deployed on Vercel
